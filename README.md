## Technology
```bash
 - NestJS - main app core
 - TypeORM - database management
 - Passport - authorization and authentication
 - bcrypt - create hash
 - class-validator - validation of all input data
 - image-size - get image size
 - sharp - resize image and change format (to .webp)
```

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
